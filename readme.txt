== Description ==

A liteweight plugin to easily display interactive 3D models on the web. Supported File type .glb, .gltf . 

**[Live Demo ](https://3d-viewer.bplugins.com/)**


= shortCode =

Use following shortCode To Embed a 3d model 

No Need to write manual shortcode by hand. You just need to choose your settings from 3D Model Settings Option and save or update then use auto generated shortcode.

1. Go to -> 3D Viewer from Dashboard menu -> Add New.
2. Configure given Options from 3D Viewer Settings.
3. Copy Generated above shortcode and use in your desired post, page and widgets. 

Enjoy 3d Viewer

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in page, post or in widgets.
4. If you want player in your theme php, Place `<?php echo do_shortcode('YOUR_SHORTCODE'); ?>` in your templates
